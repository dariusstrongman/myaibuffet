   STEP 1: Push to GitHub (2 minutes)
   ────────────────────────────────────
   cd /home/darius/Documents/myaibuffet
   
   git add .github/workflows/regenerate-seo.yml
   git add scripts/seo-enhancements.js
   git add scripts/generate-article-pages.js
   git add *.md *.txt
   
   git commit -m "SEO Enhancement: Automated regeneration workflow"
   git push


   STEP 2: Run Workflow on GitHub (3 minutes)
   ───────────────────────────────────────────
   1. Visit: https://github.com/YOUR-USERNAME/myaibuffet
   2. Click "Actions" tab
   3. Click "Regenerate Articles with SEO Enhancements"
   4. Click "Run workflow" button
   5. Type "REGENERATE" in the box
   6. Click green "Run workflow"
   
   ⏱️  Wait 5-10 minutes while it:
      • Backs up existing articles
      • Regenerates all 903 with SEO
      • Commits & pushes automatically


   STEP 3: Verify Success (2 minutes)
   ──────────────────────────────────
   1. Wait 2-5 minutes for GitHub Pages to deploy
   2. Visit: https://myaibuffet.com/articles/rss/
   3. Click any article
   4. Check for:
      ✓ "5 min read" badge
      ✓ Related articles section (3 smart matches)
      ✓ Breadcrumbs (Home → AI News → Article)
